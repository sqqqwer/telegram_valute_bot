# Telegram Valute Bot with Speech Recognition
Python telegram valute bot with parsing and speech recognition(bot work on Heroku).

Commands:
  /help - show all commands
  /btk - show bitcoin
  /usd - show usd
  /eur - show eur
  /all - show all valute
  /author - show author

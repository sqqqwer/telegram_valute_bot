# Telegram Valute Bot with Speech Recognition
Currency bot on Telegram with parsing and speech recognition (works on Heroku, language: Python).

Commands:
  /help - show all commands
  /btk - show bitcoin
  /eth - show ethereum
  /usd - show usd
  /eur - show eur
  /all - show all valute
  /author - show author
  
  
  Bot url - t.me/bitokpokazivatel_bot
